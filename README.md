# extjs-typeahead
Demo  : https://snagar78.github.io/extjs-typeahead/

# To Run Example
Clone Repository
```
    cd <folder>
    npm install
    npm start
```
Navigate to : http://localhost:9876


# Usage
```
 {
        xtype: 'textarea',
        plugins: [
            {
                ptype: 'typeAheadPlugin',
                tokenStore: 'TokenStore'
            }
        ]
}
```
